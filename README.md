## Descrição
O PokeApp é um aplicativo desenvolvido em React Native com Expo que permite pesquisar Pokémons através da PokeAPI e salvar favoritos utilizando o Firebase Firestore.

## Tecnologias
* React Native
* Expo
* Expo Router
* TypeScript
* Firebase Firestore
* PokeAPI

## Funcionalidades
* Buscar Pokémon por nome ou ID.
* Visualizar detalhes do Pokémon.
* Favoritar Pokémon.
* Editar apelido dos favoritos.
* Remover favoritos.

## API utilizada
PokeAPI: https://pokeapi.co/
